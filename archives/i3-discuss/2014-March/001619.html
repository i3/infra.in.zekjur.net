Hi Yuxuan,

Yuxuan Shui <yshuiv7 at gmail.com> writes:
> It will be better if i3 could automatically start all the programs for
> me. I think it's possible for programs that support startup
> notification.
>
> Could this feature be added, or is it too much to integrate into i3?
I’m not sure how this would work. Are you suggesting that all programs
which were opened should be started again?

This is a bit tricky. Theoretically, there is the WM_COMMAND hint in
which programs can store a command line with which they can be started
again.

However, the hint is rarely set, and when it’s set, it’s typically not
accurate.

Therefore, I currently think adding some code to use this will not be
very successful, and letting users add a bunch of “exec” lines in their
i3 config files (or session restore scripts) is acceptable.

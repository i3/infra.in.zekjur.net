Hi all,

I'd like to introduce you the first release of a new i3bar status line:
i3blocks.

The main idea behind this project is to respect the i3bar protocol [1] as
well as providing the simplest interface for the user. This means that each
block is the ouput of a shell command line.
It supports click events, signals and different time intervals, through a
simple config file.

I hope that everything is detailed on the project page:

https://github.com/vivien/i3blocks

Please let me know if you have any comment or question!

Cheers,
Vivien

[1] http://i3wm.org/docs/i3bar-protocol.html

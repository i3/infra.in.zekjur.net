Hi everyone,

I'm trying to set up colors for i3status in i3bar mode, and for some
reason it fails to work.

My i3status config file looks like this:

$ cat -n .i3status.conf
     1 general {
     2  output_format = "i3bar"
     3  colors = true
     4  color_good = "#202020"
     5  color_degraded = "#e5871f"
     6  color_bad = "#c83839"
     7  interval = 10
     8 }
     9
    10 order  = "load"
    11 order += "time"
    12
    13 load {
    14  format = "L:%5min"
    15 }
    16
    17 time {
    18  format = "%H:%M │"
    19 }

The color-related directives in my ~/.i3/config file look like this:

# -- colours -- #
#colourclass               border    backg.     text
client.focused             #1793D0   #1793D0    #FFFFFF
client.focused_inactive    #000000   #333333    #FFFFFF
client.unfocused           #000000   #333333    #FFFFFF
client.urgent              #FF0000   #FF0000    #FF0000

bar {
  status_command i3status
  colors {
    background #aeb2c3
    statusline #000000

    # color_class      border  backg.  text
    focused_workspace  #204560 #384560 #f0d830
    active_workspace   #686868 #aeb2c3 #222222
    inactive_workspace #686868 #aeb2c3 #222222
    urgent_workspace   #2f343a #aeb2c3 #ffffff
  }

But when I run i3status there are no color values in the JSON-formatted output:

$ i3status -v
i3status 59d52ad (2012-09-05) © 2008-2012 Michael Stapelberg and contributors
$ i3status
{"version":1}
[
[{"name":"load","full_text":"L:0.08"},{"name":"time","full_text":"10:42 │"}]
,[{"name":"load","full_text":"L:0.08"},{"name":"time","full_text":"10:42 │"}]
^C

Am I missing something in the configuration of i3status to actually
_send_ the colors to i3bar, like the example output in the manual of
i3bar?

I was expecting to see something like this instead, when colors = true:

$ i3status
{"version":1}
[
[{"name":"load","full_text":"L:0.08","color":
"#202020"},{"name":"time","full_text":"10:42 │","color": "#202020"}]
,[{"name":"load","full_text":"L:0.08","color":
"#202020"},{"name":"time","full_text":"10:42 │","color": "#202020"}]
^C

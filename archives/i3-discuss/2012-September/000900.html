Hi,

I would like to release i3 4.3 soon™. Therefore, it’d be very helpful,
if you all could upgrade to the latest git version of the "next" branch.
The output of i3 --version should look like this if you did it right:

    i3 version 4.2-287-ga1823e5 (2012-09-06, branch "next")

This version is stable and can be considered a release-candidate.

Also, I will not merge new features into this version until the new
release is out. You can still send your patches and I will still review
them, they just won’t be merged immediately.

If you have any bugfix or ticket that you feel is important enough to be
included in this release, please speak up now and make me aware of it.

Thanks,
Best regards,
Michael

PS: Debian and Ubuntu users can simply use our repositories for
installing the latest development version, see
http://i3wm.org/docs/repositories.html

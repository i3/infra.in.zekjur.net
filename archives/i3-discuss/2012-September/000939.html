There are two different things that are commonly done with
floating windows that get focused:

  - When focused by mouse _click_ events, raise immediately
  - When focused by mouse movement, _maybe_ raise after a timeout
(commonly called raise_on_focus)

This is currently not supported AFAIK, but if you're interested in
writing a patch, please do it :-)

The solution I've adopted for this problem is to launch a second window
manager (icewm in my case) in a different terminal. I run a little
script to do this: 
		
		xinit  /home/ac/.xxinitrc -- :1 &

where .xxinitrc is used to start icewm. I can then switch back and forth
with Shift-Alt-F8 and Shift-Alt-F7. I got this idea from ratpoison,
which has an option to do this.

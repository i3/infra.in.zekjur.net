Michael,
Sometimes a floating windows fits inside another floating window (see
attachment for an example) and if I change the focus from the top window to
any of the other two, the one that was on top is now gone. The only way for
me to access it is by moving the newly raised window to another location.
If you use a small floating window on a regular basis, like I do with a
screen ruler, and use other floating windows from time to time, this
behavior can get a bit annoying.
I still think it should be the default behavior (and I'm using it that
way). I just want a way to switch it off if I want to.

> I need change keyboard layout (becase my language is persian)

>
> best regards
> Ashkan

"setxkbmap fa" (or whatever the keyboard layout name is)
(You can put that in your xinitrc or use "exec" in your i3 config.)

You will have to adapt the bindsym statements in your i3 config though.
It may be easier to use bindcode instead of bindsym (at least if you
find yourself switching between keyboard layouts)
You can use the "xev" tool to find out the key names / key codes.

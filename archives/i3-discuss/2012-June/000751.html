On Wed, 27 Jun 2012 21:59:59 +0200
Christian Kruse <cjk at wwwtech.de> wrote:
> is there a way to move a tag from one screen to another? I've got the
> problem that all my windows are on the notebook. When I attach an
> external monitor, I want to move the whole tag (workspace) to this
> screen. Is there a solution for that?

Repeat „focus parent” until you select all windows and then move them
to another workspace like if you were to move a single window.
If you repeat the command too many times, issue „focus child” once.

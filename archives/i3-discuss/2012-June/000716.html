I've been using i3 for about a month now and I'm loving it. I was looking
for tiling manager that will manage windows as my Emacs does and i3 is the
only one I can find. I consider myself lucky that it is also stable, light
and under very active development.

However, Emacs doesn't play nice with it out of the box. So I wrote a very
basic integration layer that changes the way Emacs manages its frames so it
relies on i3 to manage window layout instead of emulating it.

I just made it available at http://github.com/vava/i3-emacs
It requires Emacs 24 to be fully functioning however i3 IPC bindings will
work fine with older versions of Emacs.

Hope it will be useful for someone else,
best regards,
Vadim.

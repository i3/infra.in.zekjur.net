The best alt-tab implementation along all the WMs was here:
http://fluxbox-wiki.org/index.php/Keyboard_shortcuts
0 or unspecified = Default/current behavior - no skipping
1 cycle groups
2 skip stuck windows
4 skip shaded windows
8 cycle linear (use the same order every time)

Of course bitmasks would have different meanings for i3 (like insetad of 
cycle windows we would cycle history or cycle linear), but the general
idea behind that is that end user configure what he thinks is more 
appropriate.

Hi,

Currently, the machine providing the autobuilder service is down. This is not a
huge deal (I can temporarily run the autobuilder on my notebook or something),
but I don’t like single point of failures and I have been looking to extend the
capacity of our autobuilder setup anyways.

So, if you want to help the i3 project, we need at least one machine with the
following specs:

 • reasonably fast CPU (anything > 2 GHz will do)
 • at least 10 GiB of disk space
 • at least 2 GiB of RAM, 4 GiB preferred (to keep the whole build chroot in
   RAM and have very fast builds)
 • fast internet connection preferred (to upload ~ 10 MiB per git commit, so
   quite low-traffic)
 • amd64 linux installation (so that we can build for i386 and amd64)

This can be a virtual machine or a physical machine, but it needs to be hosted
somewhere (that is, we are not looking for hardware donations).

Please contact me off-list if you can offer anything.

Thank you,

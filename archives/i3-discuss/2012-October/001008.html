Hi folks,

consider following layout: I've opened two windows, made the last window a split
container and then opened a 3rd window, leaving me with something like this:

  -------------------
  |       |    2    |
  |   1   |________ |
  |       |    3    |
  |       |         |
  -------------------

Consider further that I'm done with window 3, so I close it. What I now would
like to do is tell window 2 "Don't be a split container anymore!", so when I
open a new 3rd window, the 3 windows are all directly in the same container (
e.g. |1|2|3|). The only way I've found so far to do this is moving window 2
around. Is there any other way?

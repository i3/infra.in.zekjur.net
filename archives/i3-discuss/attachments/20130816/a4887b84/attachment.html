<div><br></div><div>I added in my config the option to raise and low the volume with amixer, I'm sure there must be a better way to do that, so if anyone knows better ... share it!</div><div><div><br></div><div># Section to control the Master output</div>

<div>bindSym XF86AudioRaiseVolume exec amixer set Master 1%+</div><div>bindSym XF86AudioLowerVolume exec amixer set Master 1%-</div><div>bindSym XF86AudioMute exec amixer -D pulse set Master 1+ toggle</div><div><br></div>

<div>#Section to control the Capture (mic)</div><div>bindSym Shift+XF86AudioRaiseVolume exec amixer set Capture 1%+</div><div>bindSym Shift+XF86AudioLowerVolume exec amixer set Capture 1%-</div><div>bindSym Shift+XF86AudioMute exec amixer set Capture 1+ toggle</div>

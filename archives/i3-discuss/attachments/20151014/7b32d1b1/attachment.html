    i3 tries to be smart about which windows to keep floating.<br>
    Basically, every window that specifically asks to be floating is
    kept floating.<br>
    <br>
    If you want a drastic solution, add <br>
    <pre><tt>for_window [title=".*"] floating disable
</tt></pre>
    My config tends to add floating windows :<br>
    <br>
    for_window [title="Oz Browser"]       floating enable<br>
    for_window [title="Super Hexagon"]    floating enable<br>
    for_window [title="^SpaceChem$"]      floating enable<br>
    for_window [class="Gnuplot"]          floating enable<br>
    for_window [class="FLTK"]             floating enable<br>
    #for_window [class="Graphviz"]         floating enable<br>
    for_window [window_role="CallWindow"] floating enable<br>

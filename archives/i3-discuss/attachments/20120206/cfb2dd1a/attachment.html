<br>I've implemented a Python module for communication with i3; it makes
 communicating easy by doing socket stuff for the user and it also supports subscribing to events with custom callbacks.<br>
<br>For example, the following Python code<br><blockquote>import i3<br>i3.layout('tabbed')<br></blockquote>changes the layout of the current workspace.<br><br>The user can still work with sockets and send messages to i3 as described on the <a href="http://i3wm.org/docs/ipc.html" target="_blank">IPC interface page</a>, so it also works as an i3-ipc library.<br>



There's also an i3-ipc script, which is more or less just a clone of i3-msg, and an i3wsbar implementation (still early in development though).<br><br>
Source code is hosted at <a href="https://github.com/jzib/i3-py" target="_blank">https://github.com/jzib/i3-py</a>. The <a href="https://github.com/jzib/i3-py/blob/master/README.md">README</a> contains much more information about i3-py's features.<br>

I've been using i3 for about a month now and I'm loving it. I was looking for tiling manager that will manage windows as my Emacs does and i3 is the only one I can find. I consider myself lucky that it is also stable, light and under very active development.<div>

<br></div><div>However, Emacs doesn't play nice with it out of the box. So I wrote a very basic integration layer that changes the way Emacs manages its frames so it relies on i3 to manage window layout instead of emulating it.</div>

<div><br></div><div>I just made it available at <a href="http://github.com/vava/i3-emacs">http://github.com/vava/i3-emacs</a> </div><div>It requires Emacs 24 to be fully functioning however i3 IPC bindings will work fine with older versions of Emacs.</div>

<div><br></div><div>Hope it will be useful for someone else,</div><div>best regards,</div><div>Vadim.</div>

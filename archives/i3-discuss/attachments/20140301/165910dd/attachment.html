<div dir="ltr"><div><div>Hi all,<br><br>I'd like to introduce you the first release of a new i3bar status line: i3blocks.<br><br></div>The
 main idea behind this project is to respect the i3bar protocol [1] as 
well as providing the simplest interface for the user. This means that 
each block is the ouput of a shell command line.<br>
</div><div>It supports click events, signals and different time intervals, through a simple config file.<br><br>I hope that everything is detailed on the project page:<br><br><a href="https://github.com/vivien/i3blocks" target="_blank">https://github.com/vivien/i3blocks</a><br>

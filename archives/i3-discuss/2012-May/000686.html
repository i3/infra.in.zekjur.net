Note that i3 falls back to "fixed". So it might be that you aren't
using the font that you have specified, but rather the fallback font.

I mentioned this here before, but in any case, I have patches for
Pango support in i3. Pango should render whatever you throw at it
(provided that the font supports the scripts, of course) and allow RTL
language support, besides other niceties such as antialiasing,
subpixel rendering, hinting, etc. If you or anyone else is interested,
please ping me.

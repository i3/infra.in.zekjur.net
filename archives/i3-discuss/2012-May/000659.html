Some alternatives that work well with i3:

>  3. Wicd as network manager.

Gnome's NetworkManager also works great. Invoke nm-applet from the i3
config and you'll get a tray icon (the same tray icon you get in
Gnome). I think  wicd has a GUI too, but I never tried wicd.

>  7. Parcellite.

This is great if you want to merge the X copy and paste buffer with
the toolkit's ^C + ^V buffers (Windows-like, perhaps more familiar
shortcuts). Clipboard history too.

>  12. direct control of brightness from python script

Note that this is not required for all setups.

>  13. same about sound volume

This can be done from scripts, or you can use a program like
volumeicon-alsa. With volumeicon-alsa, you also get a tray icon.

I'm totally biased here, but if you're running PulseAudio, I recommend
pa-applet. As with volumeicon-alsa, you can control the volume with
the multimedia keys in your keyboard or by interacting with the tray
icon, and you can also use the tray icon to change the audio
configuration. E.g., you can choose between HDMI or laptop speakers
using the tray icon (no need to restart programs, thanks to
PulseAudio's magic).

>  14. a set of programs to sleep, restore, add/remove external monitor

I use a script for handling other outputs. I'm not entirely happy with
it, but for the time being it works okay:

http://pastebin.com/uiF1by2w

I plan to come up with something more flexible eventually, perhaps in
the form of another tray icon. If anybody knows of an existing
solution, please let me know.

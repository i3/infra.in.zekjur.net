Hi all,
I use linux since 9 months so I am still learning new things about it
I3 was my first choice after looking at quite some number of windows
managers like ratpoison or musica etc.

To exhance i3 I use conky with dzen2. I compiled my own latest
versions (conky to avoid startup bug and dzen2 to support xft fonts).
Here are my config files: conky - http://pastebin.com/DAbJkdTK  i3:
http://pastebin.com/5a1bEehc and python script to notify about emails:
http://pastebin.com/SA4UAeiU . Note that config is designed for dvorak
for programmers keyboard layout.

Here is short list of other programs that  I use in the order of frequency:
urxvt-256color - a terminal customised to support copy-paste, link clicking etc
    config file at http://pastebin.com/xCEPvC6f
    By the way I find DejaVu Sans Mono to be the best fixed-width font
with decent unicode support.
firefox (actually iceweasel) with pentadactyl as main plugin to
minimize mouse usage. Here is my pentadactyl
config:http://pastebin.com/MSa7r8ve . Some more addons that I
sometimes use (I have several profiles) Adblock, Gnome-keyring,
FaviconizeTab, Noscript, DomainDetails, GreaseMonkey, Firebug,
Wappalyzer, Lazarus, User Agent Switcher, View Dependencies and some
other pentesting stuff.
fbxkbd - for icon for changing keyboard layout
vim - for java, latex, c++ etc (with heavy customisation) - more than
40 kb of config files :P
zsh for shell and other stuff is more common like pidgin, parcellite.

The things I am still looking for are:
- good  two panel file manager with key customization (vifm is so far
the best I found but it is terminal based and it is impossible to bind
some keys like shift tab)
- lightweiht login manager - I use gdm at the moment.
- good way to rebind keys - at the moment I am using xautomation
package but what I failed to do with it is to unify the key to exit
from applications. (like ctrl+q - but when I bind ctr+q to send ctrl+q
using xev in i3 config, bad things happen). The only solution I see is
to add window title/id bindings in i3. I have it in my todo list but
it wil for sure not have time to do it for next 3 months.
Those are my 3 cents...

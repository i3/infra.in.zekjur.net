Hi Tobias,

Quoting Tobias Eichmann (2012-05-15 18:59:36)
> exec firefox
> assign [class = "Firefox"] 1: Web
Make sure that you don’t have trailing whitespace, tabs, etc. here.

> There is no problem with that. The problem is that I can't switch to 
> workspaces named like that using the keyboard.
> I tried all of the following lines in my configuration file, but none 
> worked:
> 
> bindsym Mod4+1 workspace 1
This will not work, it looks for "1".

> bindsym Mod4+1 workspace 1: Web
> bindsym Mod4+1 workspace "1: Web"
Those should work, assuming you don’t have trailing whitespace etc. above.

> bindsym Mod4+1 workspace number 1
This has to work. At this point I suspect you are doing something very wrong,
like having two keybindings for the same key. Or not actually pressing Mod4+1.
Or modifying the wrong config file.

In any case, make sure you use a recent version of i3 (you need 4.2 for the
"workspace number" command), and provide us with i3 --version.

Also, see http://i3wm.org/docs/debugging.html in case the problem persists and
provide a compressed (!) logfile.

Best regards,
Michael

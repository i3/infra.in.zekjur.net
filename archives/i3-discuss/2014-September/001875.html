Hi.



I'm using a laptop, which is connected to two other screens
when docked. I have assigned the workspaces to the following
screens:

# eDP1 = laptop

# DP2 & DP1 = external screens.

workspace 1 output DP2

workspace 2 output DP2

workspace 3 output DP2

workspace 4 output DP1

workspace 5 output DP1

workspace 6 output DP1

workspace 7 output eDP1



When undocking I run the following randr command to switch off
the other screens:

$xrandr --output VIRTUAL1 --off --output eDP1 --mode 1920x1080
--pos 0x0 --rotate normal --output DP1 --off --output ↪HDMI2
--off --output HDMI1 --off --output DP2 --off



However I often experience that I'm unable to go to other
workspaces than 1 & 7 when undocked. Sometimes 2-6 works, but
it seems to be working quite randomly. It doesn't help either
if I log out and in again.



When searching for people having the same issue it seems that
i3 should be handling this scenario without issues, so I'm sure
there is something I have overlooked.



Any ideas on how to get this working properly?

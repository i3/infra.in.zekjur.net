"class": "^Firefox$"

But it wont swallow title as follows (notice in this case I don’t use anchors since I want to match any title containing YouTube):
"title": "YouTube"

I run firefox as follows
firefox -no-remote http://youtube.com


I've done other experiments with the "title" swallow spec, but I don’t have any luck getting i3 to swallow the title.

The complete swallows spec that I'm using is:
            "swallows": [
                {"title": "YouTube"}

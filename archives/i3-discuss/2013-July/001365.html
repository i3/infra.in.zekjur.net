When I work with Skype, I set the messaging (= text exchange) window as 
floating,
and send it to the scratchpad. When new text arrives, there is a bell 
ringing and
I call the window back from the scratchpat to answer.

The video window is always on top, but that is for the best.

So I think the scratchpad might be a solution for you.

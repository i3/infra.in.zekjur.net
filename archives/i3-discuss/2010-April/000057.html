Please enable the IPC interface (if not already done) and use the following
command (issue it on a Console or using SSH from a different computer):

  i3-msg 2

You should see i3 switching to the second workspace now. Now try clicking
on the "1" in the workspace bar. It should switch back now.

If this is the case, there is something wrong with your keybindings or
keyboard or something like that…

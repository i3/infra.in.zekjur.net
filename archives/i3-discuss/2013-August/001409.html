I dont know if this can be taken for general, but in my case giving no 
device takes the default device, which should be fine for everyone.

bindSym XF86AudioRaiseVolume  exec --no-startup-id "amixer -qM set 
Master 9%+"
bindSym XF86AudioLowerVolume  exec --no-startup-id "amixer -qM set 
Master 9%-"
bindSym XF86AudioMute         exec --no-startup-id "amixer -q  set 
Master toggle"

These are my volume bindings. They are working flawlessly.

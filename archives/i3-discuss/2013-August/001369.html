Hello,

Big user of i3 here from Canada, Quebec.
First a big thanks to the whole community for such a damn good WM.

Recently I tried to encapsulate all the key bindings of i3 in a main mode
called.... i3 which can be enabled and disabled by pressing $mod key.
The result can be seen here:
https://raw.github.com/syl20bnr/config/master/.i3/config

It has now sub modes and their names are displayed with a breadcrumb style.
It is possible to go up in the hierarchy by pressing backspace or discard
any current mode with Escape or $mod.
I read somewhere that Michael uses an ergo keyboard with some levels, I
guess this config reproduces the feeling.

The missing part is the toggle feature for the i3bar (I want to stick i3bar
when an i3 mode is enabled and hide it when leaving a mode).

Anyway being able to do it so easily in i3 is like a dream comes true. so
here is why I'm happy to share !
